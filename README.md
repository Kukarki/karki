Student Success Prediction via Data Mining. 

This project explores the relationship between gaming habits, socioeconomic factors, and student grades using the Play Games and Success in Students dataset from Kaggle.

Project Workflow:

Preprocessing: Handled "noise" (typo corrections), performed Label Encoding for categorical data, and applied Min-Max Normalization.

Data Reduction: Utilized a Correlation Heatmap to identify key attributes.

Modeling: Implemented a Decision Tree Classifier to predict student success labels.

Results: Identified Mother Education and Playing Hours as significant predictors of academic performance.
